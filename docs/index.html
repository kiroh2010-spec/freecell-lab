    게임 플레이 방법

    1. 목표: 오른쪽 위 Foundation에 A부터 K까지 같은 무늬로 쌓으면 됩니다.
    2. Tableau: 아래 카드 위에는 색이 다른 카드만, 숫자가 하나 작은 순서로 올릴 수 있습니다. 예: 검은 8 위에 빨간 7.

    5. 빠른 이동: Tableau 맨 아래 카드나 Free Cell 카드를 더블클릭하면, 가능한 경우 Foundation으로 바로 이동합니다.
