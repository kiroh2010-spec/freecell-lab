# Freecell 공유용 확장 로드맵

## 목표

현재 클릭 기반 프리셀 초안을 남들도 링크 하나로 플레이할 수 있는 웹게임으로 확장한다.

## 1단계: 플레이 감각 다듬기

- 드래그 앤 드롭 지원
- 여러 장 연속 이동 지원
- Undo / Redo
- 자동 Foundation 이동 개선
- 모바일 터치 UX 개선
- 승리 화면 추가
- 막혔을 때 안내 메시지

## 2단계: 공유 가능한 웹 버전

가장 빠른 방식:

- 정적 웹사이트로 배포
- 후보:
  - GitHub Pages
  - Netlify
  - Vercel
  - Cloudflare Pages

현재 구조가 HTML/CSS/JS라서 서버 없이 바로 배포 가능하다.

## 3단계: 게임성 추가

- 시드 기반 게임 번호
- 오늘의 도전 Daily Challenge
- 이동 수 / 시간 기록
- 최고 기록 저장
- 난이도/승률 표시
- 힌트 기능

## 4단계: 사람들에게 공유할 버전

- 이름/로고/색감 정리
- 모바일 첫 화면 최적화
- 공유용 URL 만들기
- 카카오톡/텔레그램 공유 미리보기용 메타 태그 추가
- 간단한 소개 페이지 또는 시작 화면

## 5단계: 확장 선택지

### 가벼운 웹게임 유지

- 서버 없음
- 브라우저 로컬 저장소에 기록 저장
- 배포/운영이 쉬움

### 계정/랭킹 추가

- 서버 필요
- 로그인, 리더보드, 친구 기록 비교 가능
- 운영 복잡도 증가

## 추천 방향

처음에는 서버 없는 정적 웹게임으로 간다.

1. 프리셀 룰 완성
2. 모바일 UX 다듬기
3. GitHub Pages/Netlify/Vercel 중 하나로 링크 배포
4. 반응이 있으면 Daily Challenge와 기록 저장 추가

## 바로 다음 작업 후보

1. 드래그 앤 드롭
2. 여러 장 이동
3. Undo
4. 모바일 화면 개선
5. 배포용 `public/` 구조 정리
